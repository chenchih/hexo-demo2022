  • Linux (Ubuntu, Debian)：

    $ sudo apt-get install git-core

  • Linux (Fedora, Red Hat, CentOS)

  • MAC:

    $ brew install node

  • Linux(ubuntu)

    $ sudo apt-get install -y nodejs

再安裝 Hexo

$ npm install -g hexo-cli # -g 全域安裝
